# 💡 Curriculum Proposals

이 디렉토리는 **모멘텀 리서치 팀**의 차기 커리큘럼이나 주차별 학습 자료에 대한 새로운 아이디어를 제안하고 검토하는 공간입니다. 

## 🎯 목적
- 새로운 교육 주제 및 기술 스택 제안
- 기존 커리큘럼의 개선안(난이도 조절, 실습 추가 등) 논의
- 정식 배포 전 팀원 간의 피드백 수집

## 🚀 제안 방법
1. 이 폴더에 있는 `template.md` 파일을 복사합니다.
2. 새 파일을 만들고 이름을 `prop-주제명-작성자.md` 형식으로 저장합니다.
3. 제안 내용을 상세히 작성한 후 **Pull Request**를 생성합니다.
4. 팀원들의 리뷰와 승인을 거친 내용은 `/weekly-materials`로 이동하거나 메인 레포지토리에 반영됩니다.

## 📂 파일 네이밍 규칙
- 형식: `prop-[category]-[topic]-[author].md`
- 예시: `prop-ai-llm-fine-tuning-alex.md`

## 📋 템플릿 안내
