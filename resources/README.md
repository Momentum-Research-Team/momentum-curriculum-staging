📦 Project Resources
이 폴더는 커리큘럼 구성 및 프로젝트 수행에 필요한 공통 리소스를 관리하는 공간입니다. 모든 자료는 체계적으로 분류하여 관리하며, 대용량 데이터의 경우 외부 링크나 LFS를 활용합니다.

📂 폴더 구조 (Directory Structure)
/datasets: 실습 및 분석에 사용되는 데이터셋 (CSV, JSON 등)

/templates: 노트북(.ipynb), 제안서, 보고서 등을 위한 표준 템플릿

/references: 기술 문서, 논문, 가이드라인 등 참고 자료

🛠 리소스 활용 가이드
데이터셋 추가 시: 데이터의 출처(Source)와 라이선스를 반드시 명시해 주세요.

템플릿 활용: 새로운 주차 자료를 만들 때는 /templates에 있는 표준 형식을 복사하여 일관성을 유지합니다.

참고 문헌: 유용한 블로그 포스트나 공식 문서는 링크 형태로 정리하여 /references에 추가해 주세요.

⚠️ 주의 사항
개인정보가 포함된 데이터는 절대 업로드하지 마세요.

100MB가 넘는 대용량 파일은 직접 업로드 대신 외부 스토리지 링크(Google Drive 등)를 활용하거나 Git LFS 설정을 확인하세요.
